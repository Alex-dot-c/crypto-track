# Crypto Market Tracker
A web app to track cryptocurrency prices using Flask, React, and the CoinGecko API.

## Features
- Backend: Flask API serving top 10 cryptocurrencies.
- Frontend: React app with a table displaying coin data.

## Setup
### Backend
1. Install: `pip3 install flask requests flask-cors`
2. Run: `python3 app.py`
3. Access: `http://localhost:5001/api/coins`

### Frontend
1. Navigate: `cd crypto-tracker-frontend`
2. Install: `npm install`
3. Run: `npm start`
4. Access: `http://localhost:3000`
